# sharya-academy
Run `npm install` after cloning.

## Two separate Terminal Commands to run: 
### `cd api && npm start`
Starts the Node server.

### `cd client && npm start`
Runs the React app in development mode.

## List of keys for JSON files
### Subjects:
#### `sci`: Science
#### `ssc`: Social Science
#### `mat`: Mathematics
#### `hin`: Hindi
#### `eng`: English
#### `phy`: Physics
#### `chem`: Chemistry
#### `bio`: Biology

### Content Codes:
#### `chnt`: Chapter Notes
#### `quiz`: Quizzes
#### `test`: Tests
#### `wkst`: Worksheets
#### `nsol`: NCERT Solutions
#### `xmpl`: Exemplar Questions
